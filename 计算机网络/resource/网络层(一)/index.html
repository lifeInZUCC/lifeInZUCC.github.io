    编号 选项

    B 目的主机
    C IP数据报经过的路由器
    D 目的主机或路由器

    D 211.202.59.13

    编号 选项
    A RARP
    B ICMP
    C ARP
    D IGMP

    编号 选项
    A IP路由表
    B MAC地址表
    C ARP缓存
    D 端口列表

    5.下列_______的情况下需要启动ARP请求。¶

    编号 选项
    A 主机需要接收信息，但ARP表中没有源IP地址与MAC地址的映射关系
    B 主机需要接收信息，但ARP表中只具有源IP地址与MAC地址的映射关系
    C 主机需要发送信息，但ARP表中没有目的IP地址与MAC地址的映射关系
    D 主机需要发送信息，但ARP表中只具有目的IP地址与MAC地址的映射关系

    6.下列说法错误的是_______。¶

    编号 选项
    A 通常情况下，高速缓存中的ARP是由人工建立的
    B 通常情况下，高速缓存中的ARP是由主机自动建立的
    C 通常情况下，高速缓存中的ARP表是动态的
    D 通常情况下，高速缓存中的ARP表保存了主机IP地址与物理地址的映射关系

    编号 选项
    A 传输
    B 寻径
    C 封装
    D 选择

    编号 选项
    A 纠正该IP数据报的错误
    B 将该IP数据报返给源主机
    C 丢弃该IP数据报
    D 通知目的主机数据报出错

    9._______向端系统提供虚电路和数据报两种网络服务。¶

    编号 选项
    A 广域网
    B 通信子网
    C 资源子网
    D 局域网

    10.通常分配给小型网络使用的IP地址是_______类地址。¶

    编号 选项
    A A
    B B
    C C
    D D

    11.三层交换机在转发数据时，可以根据数据包的_______进行路由的选择和转发。¶

    编号 选项
    A 源IP地址
    B 目的IP地址
    C 源MAC地址
    D 目的MAC地址

    编号 选项

    B 源地址有可能发生变化，而目的地址不会发生变化
    C 源地址不会发生变化，而目的地址有可能发生变化
    D 源地址和目的地址都有可能发生变化

    编号 选项

    C 默认路由
    D RIP路由

    编号 选项

    D 可靠，无连接和全双工

    编号 选项

    B 面向连接的不可靠的服务
    C 无连接的可靠的服务
