    应用层(一)

    1.以下关于应用层协议基本内容的描述中错误的是________。¶

    编号 选项
    A 定义了为交换应用进程数据提供服务的传输协议报文格式
    B 定义了各种应用报文的格式与字段类型
    C 规定了每个字段的意义
    D 对进程在什么时间、如何发送报文，以及如何响应进行了描述

    2.以下关于域名解析的描述中错误的是________。¶

    编号 选项
    A DNS协议可以完成物理地址解析
    B 递归查询由本地域名服务器负责解析的全过程，只将最终结果返回给客户端
    C 迭代查询由客户端解析程序负责多次解析请求的全过程
    D 实际在软件编程中人们主要采用在客户端的迭代查询的方法

    3.以下关于TELNET基本工作原理的描述中错误的是________。¶

    编号 选项
    A TELNET服务采用C/S工作模式
    B NVT能够解决不同终端与主机之间通信兼容性问题
    C TELNET主机与终端之间传输的数据都是标准的ASCII格式
    D TELNET客户端与服务器进程完成终端、主机内部格式与NVT格式之间的转换

    4.以下关于HTTP协议持续连接特点的描述中错误的是________。¶

    编号 选项
    A 持续连接有两种工作方式：非流水线与流水线
    B HTTP1.0默认状态是持续连接的流水线工作方式
    C 非流水线方式是：客户端只有在接收到前一个响应时才能发出新的请求
    D 流水线方式是：客户端与服务器端可以连续地发送请求与应答报文

    5.为了使客户端程序查询不同的信息资源时有统一的访问方法而定义的一种地址标识是________。¶

    编号 选项
    A URL
    B HTML
    C HTTP
    D WWW

    6.HTTP是________。¶

    编号 选项
    A 超文本标记语言
    B 超文本传送协议
    C 搜索引擎
    D 文件传输协议

    7.域名服务DNS的解析是________。¶

    编号 选项
    A 将域名转换为物理地址
    B 将域名转换为IP地址
    C 将IP地址转换为物理地址
    D 将IP地址转换为域名

    8.网络中应用层的许多协议都是基于________。¶

    编号 选项
    A 客户服务器方式
    B 客户与客户方式
    C 服务器与服务器方式
    D 应用程序与应用程序方式

    9.在网络通信中服务器是________。¶

    编号 选项
    A 服务请求方
    B 服务提供方
    C 请求发送方
    D 请求提供方

    10.因特网中域名标号不能超过________。¶

    编号 选项
    A 32个字符
    B 64个字符
    C 63个字符
    D 128个字符

    11.以下不是FTP的主要功能有____________。¶

    编号 选项
    A 实现文件的上传与下载
    B 提供对本地计算机和远程计算机的目录操作功能
    C 传输多种类型、多种结构、多种格式的文件
    D 动态分配客户端的IP地址

    12.使用FTP进行文件传送时，使用________端口。¶

    编号 选项
    A 21一个
    B 20一个
    C 20与21二个
    D 21与80二个

    13.使用FTP进行数据传送时，运输层中使用___________协议。¶

    编号 选项
    A UDP
    B TCP
    C PPP
    D 任何通信协议

    14.Telnet指的是______。¶

    编号 选项
    A 万维网
    B 电子邮件
    C 文件传输
    D 远程登录

    15.万维网的全称是________。¶

    编号 选项
    A Wide World Web
    B World Wide Web
    C Wide Web World
    D Web Wide World

    16.第一个图形界面的浏览器是__________。¶

    编号 选项
    A Internet Explorer
    B Netscape Navigate
    C Firefox
    D Mosaic

    17.万维网中传送到用户的文件是________格式文件。¶

    编号 选项
    A HTML
    B WORD
    C PPT
    D EXCEL

    18.HTTP的URL一般形式是________。¶

    编号 选项
    A http：//<主机>：<端口>/<路径>
    B http：//<主机>：<路径>/<端口>
    C http://<路径>：<主机>:<端口>
    D http://<端口>/<路径>/<主机>

    19.用户上网输入网址浏览网页时，可省略的是________。¶

    编号 选项
    A 主机名
    B 端口号
    C 域名
    D 二级域名

    20.超文本标记语言HTML扩展名为________。¶

    编号 选项
    A TXT
    B HTM
    C DOC
    D EXE
