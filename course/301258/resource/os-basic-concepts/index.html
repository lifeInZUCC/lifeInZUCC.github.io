    操作系统基本概念

    1.操作系统功能不包括( )。¶

    编号 选项
    A 提供用户操作界面
    B 提供 HTML
    C 提供应用程序接口
    D 管理系统资源

    2.多道程序设计是指( )。¶

    编号 选项
    A 在多台处理机上同时执行多道程序
    B 在多台处理机上同一时刻执行多道程序
